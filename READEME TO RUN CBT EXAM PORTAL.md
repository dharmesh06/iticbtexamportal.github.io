Step 1: Start your application or web server



&#x09;**Method-1: (if Node.js already Installed)**

&#x09;open Command Prompt

&#x09;cd \[path Of Your Project Folder]

&#x09;Type this command--> **http-server -p 8080**

&#x09;Now Double Click on index.html (Your Webpage live)

&#x09;

&#x09;**Method-2:(if Python is already installed)**

&#x09;Type this command-->**python -m http.server 8080**

&#x09;Now Double Click on index.html (Your Webpage live)



&#x09;**Method-3:(if Docker is already installed)**

&#x09;Type this command-->**docker run -p 8080:80 my-image**

&#x09;Now Double Click on index.html (Your Webpage live)



Step 2: Access in a web browser

&#x09;Open your web browser (Chrome, Firefox, Edge, etc.) and type 	the following address into the address bar: 	

&#x09;**http://localhost:8080**



&#x09;
